Сайт для Федерации Компьютерного Спорта Республики Коми.

На сайте у всех желающих будет возможность зарегистрироваться на турнир по
таким играм, как Dota2, Counter-Strike2, Starcraft и другим.

Также на сайте будет присутствовать возможность смотреть стримы основных турниров.

При этом на сайте пользователь сможет ознакомиться с актуальными новостями киберспорта.


Техническое Задание(ТЗ)
1. Визуальная часть:
   1) Разработать модель всех страниц сайта в цветовом редакторе(painte)
   2) Найти предпочтительные элементы сайта на Flowbite
   3) Выбрать цветовую гамму и при помощи классов tailwind добавить дизайн на сайт
2. База Данных:
   1) Разработать БД для хранения информации о зарегистрированных пользователях
3. Регистрация/вход пользователя на сайт:
   1) Разработать форму регистрации и входа
   2) Соединить с БД
4. Регистрация на турниры:
   1) Разработать форму регистрации для турниров
5. Новости:
   1) Сделать страницу новостей
   2) Добавить возможность обновлять новости администраторам
6. Трансляции:
   1) Добавить возможность пользователям смотреть трансляции идущих турниров
7. Связаться с заказчиком и предложить сайт))

Пояснительная записка(ПЗ)
Название проекта: cybersport
Авторы проекта: Зайцев Сергей Петрович и Калимов Денис Сергеевич
Описание идеи: Проект представляет собой функциональную веб-платформу для подростков, желающих принимать участие в сфере киберспорта в Республике Коми. Она обеспечивает быструю и эффективную коммуникацию с организаторами турнира - Федерацией компьютерного спорта РК, а также внутри сообщества. При это сайт также обеспечивает втоматизацию процесса записи на соревнования, возможность просмтора трансляций турниров и многое другое.)
Описание реализации и технологий: Проект основан на базе данных, хранящей информацию о зарегистрированныхна турнир пользователях. Все взаимодействие с БД проиходит в обработчиках Flask. Реализованы различные формы с помощью flask-wtf и WTForms. Для взаимодействия клиента с сервером были созданы обработчики POST и GET. Необходимые библиотеки для запуска: flask, flask-login, flask_wtf, wtforms, sqlalchemy.
